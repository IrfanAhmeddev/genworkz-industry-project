# Genworkz Industry Project 🚀

## Overview
This is an industry-oriented full stack web development internship project built collaboratively by students using GitHub workflow.

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express.js or Supabase
- Database: Supabase DB
- Tools: Git, GitHub, Supabase authentication, Supabase Api

## Project Structure
frontend/   → UI code  
backend/    → Server & APIs  
database/   → postgreSQL schema  
docs/       → Documentation  

## Rules
- No direct commits to main branch
- Work only in feature branches
- Raise Pull Requests for merging
